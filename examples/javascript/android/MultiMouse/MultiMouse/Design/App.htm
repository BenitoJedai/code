MultiMouse

Once this window locks the mouse pointer it will be shared between devices.

Notice the cursor is svg. Drop shadow is actualy glow with offset filter.

Ther is a big svg canvas here to act as a virtual screen you can draw upon which can also be synchronized.

Screen Configuration

In this mode you can rearrange the position of your screens. Screen is informative.

A child window on the same device can enable multiple screen scenario.
