Who are we?

We provide the converson from a .NET language to another non .NET language for you.!

The team behind jsc compiler at this time consists of two people: Arvo Sulakatko (Estonia) and Carlo Godoy (Canada)

How does it work?

The jsc compiler will read the IL bytecode and emit source code. Additionally the jsc compiler will use compilers like mxmlc and javac and use their output in a way to make them available for JavaScript in the Browser Application Languages supported:
  • JavaScript
  • Java
  • ActionScript
  • PHP

Where are we going?

In our vision the developers will be able to write in .NET in C# for any platform.
