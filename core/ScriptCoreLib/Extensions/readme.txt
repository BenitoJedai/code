[ScriptCoreLib] ScriptCoreLib.Extensions

Store extension methods that will be available on all platforms,
are precompiled and cached.

Classes should use prefix Core.
Classes need to use [Script] attribute to be compiled.

Those methods defined can be used within the ScriptCoreLib framework code.

[ScriptCoreLib.Extensions] ScriptCoreLib.Extensions

Store extension methods that will be available on all platforms,
but are not precompiled and cached and are linked only if used.

Those methods defined can be used within the application code.
