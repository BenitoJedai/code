FlashApplication. Crosscompiled from c# to actionscript.

Game Description:
A remake of a classic electronic puzzle game.
Pressing one of the lights will toggle it and the four lights adjacent to it on and off. 

Game Instructions:
Try to turn off all the lights by clicking on them.

Category:
Puzzle

Steps to create a Nonoba Multiplayer game out if this game:
01. Add new LightsOut.Client project to the solution
02. Reference it with this game
03. Change project default namespace
04. Add Client folder and TeamPlay.cs
05. Update build.bat for TeamPlay
06. Add reference to ScriptCoreLib.Nonoba and ScriptCoreLib.Mochi
07. Add SharedClass1
08. Add update.bat
09. Run update and include SharedClass1.Generated.cs
10. Define some basic network messages
11. Add predefined file: Generic/ServerGameBase.cs
12. Add predefined file: Generic/ServerPlayerBase.cs
13. Add predefined file: Player.cs
14. Add predefined file: Game.cs
15. Add predefined file: IDispatchHelper.cs
16. Add predefined file: IEventsDispatch.cs
17. Update namespaces
18. Add extension methods
19. Add sound assets
20. Compile LightsOut.Client
21. Disable LightsOut build.bat (goto :skipped)
22. Add new LightsOut.Server project to the solution
23. Add predefined file: Player.cs
24. Add predefined file: Game.cs
25. Update namespaces
26. Add Library files
27. Add reference to Nonoba.GameLibrary.dll
28. Create folder System and add files
29. Create folder Generic and add files
30. Create folder Shared and add files as links from LightsOut.Client
31. Create folder Shared/Generic and add files as links from LightsOut.Client
32. Add conditional flag "NoAttributes"
33. Add new LightsOut.ServerTest project to the solution
34. Set LightsOut.ServerTest as startup project
35. Add references
36. Compile and test
37. Add game state syncing on join
38. Add build event to copy swf and dll for Nonoba
39. Add coplayer mouse
40. Create a preview image 100x100
41. Setup MochiAds entry
42. Add MochiPreloader.cs
43. Update game_id
44. Update build.bat
45. Compile and run MochiPreloader swf once
46. Start filling out Mochi Game profile
47. Setup Nonoba game account
48. Link Mochi to Nonoba
49. Download Nonoba version and upload it to Mochi for distribution
50. Create a ranking for games played.
51. Add network message for it.
